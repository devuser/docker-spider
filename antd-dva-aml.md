反洗钱系统前端
==============

技术栈
------

1.	Docker
2.	Node.js
3.	Yarn
4.	Antd
5.	React.js

脚手架
------

1.	安装`yarn` `npm install -g yarn`
2.	安装`dva` `yarn global add dva-cli`
3.	创建项目 `dva new antd-dva-aml;cd antd-dva-aml;`
4.	安装依赖包 `npm install`
5.	启动 `npm start`
6.	Docker容器内访问 `http://127.0.0.1:48000` 主机映射到`48000`访问 `http://127.0.0.1:48000`
